# CBN Risk-Based Cybersecurity Framework Checklist

**Reference:** CBN Risk-Based Cybersecurity Framework and Guidelines, issued 31 May 2024, effective 1 July 2024
**Applies To:** Commercial banks, merchant banks, non-interest banks, and Payment Service Banks (PSBs)
**Prepared by:** TrustGrid Technology

---

## How to Use This Checklist

Work through each section with your Chief Information Security Officer and compliance team. Mark each item as complete, in progress, or not started. Every gap identified here is a real regulatory exposure, this framework is actively enforced, not advisory.

**Status Key**
- Complete
- In Progress
- Not Started

---

## Section 1: Governance

| # | Requirement | Status |
|---|---|---|
| 1 | A named Chief Information Security Officer (CISO) has been appointed | |
| 2 | The CISO reports directly to the Board or a Board-level committee | |
| 3 | The CISO role is distinct from IT operations, avoiding conflict of interest | |
| 4 | A documented cybersecurity policy approved by the Board is in place | |
| 5 | Cybersecurity risk appetite has been defined and approved by the Board | |
| 6 | The Board receives regular cybersecurity risk reporting in business terms | |

## Section 2: Risk Assessment and Management

| # | Requirement | Status |
|---|---|---|
| 7 | A formal cybersecurity risk assessment has been conducted and documented | |
| 8 | Risk assessment results are integrated into the enterprise risk register | |
| 9 | Identified risks have assigned owners and treatment plans | |
| 10 | Risk assessments are reviewed and updated on a defined schedule | |

## Section 3: Third-Party and Vendor Risk

| # | Requirement | Status |
|---|---|---|
| 11 | All vendors with system or data access have been identified and inventoried | |
| 12 | Vendor security assessments are conducted before onboarding | |
| 13 | Contracts with vendors include security and breach notification clauses | |
| 14 | Ongoing vendor risk monitoring is in place, not just a one-time assessment | |
| 15 | Outsourcing arrangements have been assessed against CBN outsourcing guidelines | |

## Section 4: Emerging Technology

| # | Requirement | Status |
|---|---|---|
| 16 | AI systems in use have been identified and risk-assessed | |
| 17 | Cloud computing arrangements have documented security controls | |
| 18 | Distributed ledger technology (DLT) use, if any, has been assessed for risk | |
| 19 | Emerging technology risks are reported to the Board alongside traditional risks | |

## Section 5: Technical Controls

| # | Requirement | Status |
|---|---|---|
| 20 | Multi-factor authentication is enforced for all privileged and remote access | |
| 21 | Network segmentation is implemented and documented | |
| 22 | Endpoint detection and response tools are deployed | |
| 23 | Regular vulnerability assessments and penetration testing are conducted | |
| 24 | Patch management processes are documented and followed | |
| 25 | Data encryption is enforced for data at rest and in transit | |

## Section 6: Incident Response

| # | Requirement | Status |
|---|---|---|
| 26 | A documented incident response plan exists and has been tested | |
| 27 | Incident notification procedures to the CBN are documented | |
| 28 | Staff know how to report a suspected security incident | |
| 29 | Post-incident review processes feed back into risk management | |

## Section 7: Regulatory Interconnection

| # | Requirement | Status |
|---|---|---|
| 30 | Cybersecurity controls are aligned with BOFIA 2020 requirements | |
| 31 | Data protection controls are aligned with the NDPA 2023 | |
| 32 | The organisation is registered with the NDPC where applicable | |
| 33 | Annual Compliance Audit Return obligations under the NDPA are tracked separately from CBN reporting | |

## Section 8: Audit and Examination Readiness

| # | Requirement | Status |
|---|---|---|
| 34 | Documentation is organised and readily available for CBN supervisory review | |
| 35 | The organisation has conducted an internal readiness assessment against this framework | |
| 36 | Previous CBN examination findings, if any, have been remediated and closed | |
